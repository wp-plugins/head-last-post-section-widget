Select a post or page and display the first "section"(as you put the delimiters) in a widget.


== Installation ==
1. Install from Wordpress REPO.
2. Go to Widgets and add a widget!

== Description ==

With this Widget-Plugin you can add a widget to your wordpress and select a post or page.
Then you insert a regular expression as provided to preg_split php function in order to
split your page to "sections". Then you select how many "sections" the widget will display.
